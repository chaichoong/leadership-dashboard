🛠 1. Overview

The Contractor Job List provides a filtered view of all maintenance and repair tasks that have been assigned to external contractors. It pulls from the same Airtable Tasks table used across the Operations Director base, but filters specifically for records where the Maintenance Ticket checkbox is ticked.

This view is designed for quick reference during contractor calls, site visits, and weekly job reviews.

ℹ️ Jobs are created via the Contractor Job Creator skill in Claude Code or manually in Airtable. Both methods tick the Maintenance Ticket checkbox automatically.

🗃 2. Data Source

Airtable Configuration
  • Base: Operations Director
  • Table: Tasks
  • Filter: Maintenance Ticket = true
  • Sort: By status (active jobs first), then by due date ascending

Key fields displayed: Task Name, Property, Contractor, Status, Due Date, Priority, Description, and Cost Estimate.

🔗 3. How to Access

  1. Open the Leadership Dashboard in your browser.
  2. Click the Contractor Jobs tab in the navigation bar.
  3. The job list loads automatically, showing all active maintenance tickets.

👁 4. Using the View

What You Can Do
  • View job details — Click any row to expand and see the full description, contractor notes, and cost estimate.
  • Filter by property — Use the property dropdown to narrow the list to a specific address.
  • Filter by status — Toggle between Active, Completed, and All jobs.
  • Sort columns — Click column headers to sort by due date, priority, or contractor name.
💡 To add a new contractor job, use the Claude Code command "add a job" or create a record directly in the Airtable Tasks table with the Maintenance Ticket checkbox ticked.

🔧 5. Troubleshooting

Job not appearing in the list
Verify the Maintenance Ticket checkbox is ticked on the Airtable record. If the job was just created, refresh the dashboard page to pull the latest data.
Completed jobs still showing as active
Check the task Status field in Airtable. The dashboard filters by status value, so ensure the status has been updated to "Completed" (not just marked done in comments).
Property column showing blank
The Property field is a linked record. Ensure the task record is linked to the correct property in Airtable. A missing link results in a blank cell on the dashboard.
