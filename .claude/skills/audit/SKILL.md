# Audit & Score
Run a robustness audit on the specified page/dashboard:
1. Test the live site (auth-permitting) and code-level checks
2. List bugs found with severity
3. Fix each issue and commit with descriptive messages
4. Re-audit to catch self-introduced bugs (badge counts, filter logic, double-submits)
5. Push to GitHub and verify deploy is live
6. Provide saleability/readiness score out of 100
