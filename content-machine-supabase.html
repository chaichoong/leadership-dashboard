🌐 Insta360 Reframe
⚠️
What this tool can and cannot do vs Insta360 Studio
✅ What it CAN do
Drag canvas to pan/tilt the 360° view
MEGA / ULTRA / DEWARP / LINEAR presets
Pan · Tilt · Roll · FOV sliders
Manual keyframe placement (+button)
Click yellow dots to jump to keyframes
Linear interpolation between keyframes
Works on exported flat MP4 files
❌ What it CANNOT do
No AI subject tracking — Insta360 Studio's DeepTrack is a proprietary GPU model that auto-centres Kevin on every frame. This requires a native app with GPU access. A browser cannot do this at video speed.

No raw .insv files — browser can't decode Insta360's format. Only exported MP4 works.

No Bezier curves — Studio interpolates per-frame with smooth curves. This tool uses linear interpolation at 30fps.

No ProRes output — browser export caps at ~8 Mbps. Studio can do 200+ Mbps ProRes.
👉 Recommendation: Use Insta360 Studio (free at insta360.com/download) for the reframe step — it auto-centres Kevin properly. Export a flat MP4, then bring it here for the rest of the workflow.
