1 Purpose

This operating system guides a client from a rough business idea to a complete, structured business plan and 30-day launch action plan through an AI-powered conversation. The AI coach asks questions one at a time, extracts the information naturally, and compiles everything into a professional document with PDF export.

2 Overview

  • Type AI-guided conversational tool
  • Technology Single-file HTML/CSS/JS application
  • AI Provider Claude (Anthropic) or ChatGPT (OpenAI) or Custom endpoint
  • Data Storage Browser localStorage (client-side only)
  • Output 10-section business plan + 30-day launch plan with PDF export
  • Requirements No installation required — runs in any modern web browser

3 Setup Instructions

For Operations Director Team (Deployment)

  1. The operating system is hosted on GitHub Pages
  2. Access it at: chaichoong.github.io/leadership-dashboard/os/business-plan-builder/
  3. For local testing: serve via Python HTTP server through the proxy server (serve.py)

For Clients (End Users)

  1. Open the Business Plan Builder link in your browser
  2. Choose your AI provider (Claude recommended)
  3. Enter your API key (get one from console.anthropic.com)
  4. Type your name
  5. Click "Start Building Your Plan"
  6. Answer the AI coach's questions naturally — one at a time
  7. Watch sections get completed in the sidebar as you go
  8. Click "View Your Plan" at any time to see your progress
  9. When all 10 sections are done, export as PDF
Tip

Your progress is automatically saved in the browser. You can close the tab and come back later — everything will still be there.

6 Technical Architecture

  • Codebase Single HTML file (~1500 lines) with inline CSS and JavaScript
  • API Routing API calls routed through local proxy server (serve.py) to avoid CORS
  • Proxy Support Supports Anthropic, OpenAI, and custom API endpoints
  • Data Extraction Plan data extracted from AI responses using [PLANDATA:section_id]{JSON}[/PLANDATA] markers
  • Persistence State persisted to localStorage under key od-bplan-v4

7 Troubleshooting

"Failed to fetch"

Check the proxy server is running. Make sure your API key is valid and has not expired.

"Could not connect to AI"

Verify your API key is correct. Check billing is active on your API provider account (Anthropic or OpenAI).

Stuck on setup screen

Clear your browser data for this site, or add ?reset=1 to the end of the URL to force a fresh start.

Lost progress

Data is stored in your browser's localStorage. Using a different browser or device won't have your saved progress. Stick to the same browser you started with.
